What is TidyTuesday?

TidyTuesday is a weekly data visualization and analysis challenge designed to encourage the use of R and the tidyverse. Each week, participants work with publicly available datasets to create compelling visuals, uncover insights, and improve their data wrangling and storytelling skills while sharing their work with the community.
